🛡️ About ScamShield
ScamShield uses intelligent pattern detection to identify SMS and WhatsApp scams in real-time. Paste a suspicious message and get instant feedback: Is it a scam? Why? What should you do?
Built for Malaysians. Designed to recognize Maybank alerts, BNM scams, investment fraud, and job offer scams.
📖 How It Works
  1. Paste a Message - Copy the suspicious SMS or WhatsApp text
  2. Analyzes - Scam pattern detection for common Malaysian frauds
  3. Get Results - See risk score, flags, and explanation
  4. Take Action - Delete, block, or report the message
⚠️ If You've Been Scammed
🚨 Act Immediately:
1. Contact your bank immediately
Maybank: 1-300-88-6688
CIMB: 1-800-811-125
RHB: 1-800-811-098

2. File a report with PDRM (Police)
3. Report to BNM Complaint Bureau
BNM: 1-300-880-088

4. Do NOT send more money
5. Save all evidence (screenshots, transaction details)
💡 Stay Safe: Tips
✓ Real banks NEVER ask for passwords or OTP via message
✓ Verify by calling the bank directly (use the number on the back of your card)
✓ Hover over links to see the real URL (don't click shortened links)
✓ Watch for poor grammar and spelling (common in scams)
✓ If something feels off, trust your gut - ask a friend or family member
